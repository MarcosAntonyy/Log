    * Teoria dos Conjuntos *

      * É o assunto que estuda-se os "conjuntos" no universo, esta teoria foi desenvolvida pelo matemático alemão Georg Cantor(1845 - 1918) no final do século XIX através da necessidade de pesquisar mais acerca do finito e infinito e as propriedades dos números e a "cardinalidade" dos objetos de qualquer natureza nessas coleções.

      ** Conjuntos **

        * São coleções de objetos ou elementos no universo de formas abstratas ou concretas de várias maneiras porém com exclusividade o sistema de chave e valor "Conjunto = { Elemento(s) }".

        * Tipos de Conjuntos

        • Conjunto Vazio: Não possui elementos, ∅ = {};

        • Conjunto Finito: Possui uma quantidade finita (esgotável) de elementos, F = {a, b, c, ◯};

        • Conjunto Infinito: Possui uma quantidade infinita (inesgotável) de elementos, I = {a, b, c, ◯,...};

        • Conjunto Unitário: Possui apenas um elemento, U = {y};

        • Conjunto Universal: Possui todos os elementos de um escopo ou universo considerado, Área = {Subáreas};

        • Conjunto Numérico: Possui todos os números divididos em categorias especifícas na Aritmética, N (números naturais), Z (números inteiros), Q (números racionais), R (números reais) e C (números complexos);


        * Relações de Conjuntos e seus Elementos

        • Pertinência ( ∈ ): Indica os elementos de um conjunto presentes em outro(s);

            > (∈)  | X = {g, h, i} e Y = {a, b, g}, então X ∈ Y |, porque pelo menos um elemento de X pertence a Y.
            > Não Pertinência (∉) | X = {j, k, l} e Z = {a, 7, 0}, então x ∉ Z|, porque nenhum de X pertence a Z.


        • Igualdade ( = ): Indica conjuntos que possuem elementos idênticos;

            > (=) |X = {1, 4, 5} e Y = {1, 4, 5} ou objeto, então X = Y e objetos = objetos|, porque todos são idênticos por natureza.


        • Inclusão, "está contido em" ( ⊂ ): Indica conjuntos que estão contidos em outros com seus respectivos elementos;

            > ( ⊂ ) | X = {8, 9, 0} e Y = {8, 9, 10, 11}, então X ⊂ Y|;
            > Não Inclusão (⊄), "não está contido em" |A = {a, y, r} e B = {a, e, i, o, u}, então A ⊄ B|;
            > ( ⊃ ) Próprio, "contém" ou "supremacia de"  | A = {a, e, i} e B = {a, e, i, o, u}, então A ⊃ B|;
            > Subconjunto (⊆), "está contido ou é igual a"   |A = {1,2,3} e B = {1,2,3,4,5}, então A ⊆ B|;
            > Não Subconjunto (⊈), "não está contido ou não é igual a"  |A = {j, k, l} e B = {a, b, c, d, e}, então A ⊈ B|;


        • União, "e" ( ∪ ): Indica conjuntos de elementos que juntam-se dois ou mais conjuntos de elementos sem duplicar os elementos.

            > ( ∪ ) | X = {a, b, c, d, e} e Y = {1, 2, 3, 4, 5, 6}, então A ∪ B = {a, b, c, d, e, 1, 2, 3, 4, 5, 6} |;
            > União de Múltiplos Conjuntos  | A = {1,2}, B = {5,6} e C = {0,9}, então A ∪ B ∪ C = {0, 1, 2, 5, 6, 9} |;
            > União Infinita  | ⋃{A i} i ∈ I |;
            > União Disjunta  | A = {7, 6} e B = {8, 9}, então A ⊎ B = {(7, A), (6, A), (8, B), (9, B)} |;
            > União Simétrica (△)  | A = {1, 2, 3} e B = {3, 4, 5}, então A △ B = {1, 2, 4, 5} |;


        • Interseção ( ∩ ): Indica conjuntos de elementos que são comuns em outros conjuntos ou que pertencem a dois simultâneamente.

            > (∩)  |A = {1, 2, 3, 4} e B = {3, 4, 5, 6}, então A ∩ B = {3, 4}|;
            > Interseção Nula  |A = {○, □} e B = {Δ, X}, então A ∩ B = ∅|;

        ** Diagrama de Venn-Euler **

           Foi desenvolvido pelo matemático britânico John Venn que em 1880 desenvolveu essa representação gráfica para ilustrar as relações lógicas entre diferentes conjuntos, com a ideia de facilitar a visualização de como os conjuntos de informações se intersecam e se relacionam entre si, sendo uma ferramenta útil em várias aplicações com informações em desenho.



        ** Características dos Conjuntos **

          1. Elemento: É necessário que objetos façam parte do conjunto para ser considerado um.

          2. Pertinência: Um elemento pertence "∈" ou não pertence "∉" a um conjunto.

          3. Definição: Um conjunto deve determinar de maneira clara se um objeto pertence ou não a ele.
