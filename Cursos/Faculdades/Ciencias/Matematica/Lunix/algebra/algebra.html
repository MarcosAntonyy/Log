| # Álgebra #

* O que é?

      Do árabe, al-jabr(restauração ou união), esta estuda-se a matemática de forma alfanumerica generalizando a aritmética abstrata e concreta.

* Escopo dos Assuntos


    * Algebra Elementar (Básica) *

      * É o assunto da Álgebra que estuda-se os fundamentos algébricos, desenvolvida conforme a história das civilizações antigas e principalmente através dos matemáticos Diofanto de Alexandria com sua obra "Arithmetica", o persa Al-Khwarizmi (780 - Bagdá, c. 850) que originou o nome "álgebra" pela sua obra "Kitab al-Jabr wa'l-Muqabala", e René Descartes (1596 - 1650) com suas colaborações aplicadas da Álgebra na Geometria.

      ** # Aritmética # **


      ** Expressões Algébricas **

        * Elementos das Expressões

        • Variáveis Algébricas: São números desconhecidos (icógnitas) geralmente representadas por objetos ou caracteres, "□ ou letras";

        • Constantes Algébricas: São números fixos dados conforme a questão, "números (R)";

        • Coeficientes Algébricos: São constantes multiplicadas por variáveis, "3□ ou 2x";

        • Termos Algébricos: São partes da expressão (constantes e coeficientes) separadas por operações aritméticas, "5□ + 2 ou 2x + 4, que são consideradas compostas por 2 termos, 5□ e 2 ou 2x e 4";


        * Propriedades Álgebricas

        • Comutatividade:
            > Adição (+), a + b = b + a;
            > Multiplicação (x), a x b = b x a;

        • Associatividade:
            > Adição (+), (a + b) + c = a + (b + c);
            > Multiplicação (x), (a x b) x c = a x (b x c);

        • Distributividade:
            > a x (b + c) = a x b + a x c

        • Identidade:
            > Adição (+), a + 0 = a;
            > Multiplicação (x), a x 1 = a;

        • Inverso:
            > Adição (+), a + (-a) = 0;
            > Multiplicação (x), a x 1/a = 1;


        * Tipos de Expressões Algébricas

        • Monôminais

        • Binôminais

        • Polinômiais

        • Racionais

        • Irracionais
