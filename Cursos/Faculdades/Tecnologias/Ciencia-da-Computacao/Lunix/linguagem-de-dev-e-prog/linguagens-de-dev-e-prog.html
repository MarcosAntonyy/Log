* O que é?

      São linguagens utilizadas na Ciência da Computação como fundamento de comunicação entre humano e máquina no desenvolvimento e programação de softwares conforme as suas categorias e terminologias no mercado de trabalho e as mesmas possuem níveis de uso, sendo "baixo nível" a qual se aproxima da interpretação das máquinas que é de base binária ()2 "0 e 1" e as de "alto nível" que tem por afinidade em linguagem natural ou "linguagem humana".

    * Linguagens de Desenvolvimento e Programação Científica *

      *   São linguagens de desenvolvimento e programação exclusivas para aplicações científicas que por vezes serve como base para outras funções em linguagens de desenvolvimento e programação.



      * Linguagens Científicas

      •   &  

          *   São linguagens de programação que visam a aplicação da ciência matemática com suas funções em essência, sendo ela uma linguagem interpretada possui um uso multiplataforma com compatibilidade entre as duas e em outros dispositivos.



          * Escopo de Sintaxe

          • Comandos Base

              > clear nomedavar / clearvars / all (Limpa a todas as variáveis da memória)


              > clc ( Limpa a janela de comandos )


              > close all ( Fecha todas as janelas gráficas )


              > ; (impressão de ação executada em linha ou supressão)


              > version ( Mostra a versão do pacote MathLab / Octave )


              > format long / short (Regula o número de casas decimais sendo impressos ou guardados)


              > disp (x) ( Imprime o conteúdo na tela )



          • Operadores

              > Aritmético ( + , - , * , /, ^ )


              > Relacional ( <, >, <, =, >= )


              > Lógico ( ==, !=, !&, |, xor, ->, 0 (~) )



          • Funções

              > Funções Base

                - sqrt(x) ( Raiz quadrada de x)
                - log(x) (Logaritimo Natural de x, para modificar a base basta inserir um número antes do parenteses ex: log10(x) )
                - factorial(x) (Fatorial de x!)
                - rem(x, y) (O resto da divisão de x por y, % também faz a mesma coisa)
                - abs(x) (Valor absoluto ou módulo)
                - round(x) (Arredonda um valor)
                - fix(x) (Trunca um valor)
                - floor(x) (Arrendonda ao inteiro inferior)
                - ceil(x) (Arrendonda ao inteiro superior)
                - e^x == exp(x) (Elevar um número a constante de euler)
                - tik e tok (Inicia e finaliza um contador que se atribuido a uma variavel pode ser feito de cronomêtro)

              > Funções Intermédiarias

                - A = [ x, y, z; p, q , r] (Declaração de uma matriz, o ; encerra onde seria a linha, essa matriz no exemplo é 2x3)
                - size(x) (Retorna o número de linhas e colunas de uma matriz x)
                - size(x, 1) ou size(x, 2) (Retorna o número de linhas (1) ou o número de colunas (2))
                - lenght(x) ( Retorna o maior comprimento da matriz x ou de um vetor x)
                - eye(x) ( Cria uma matriz identidade de ordem x)
                - trace(x) ( Soma os elementos da diagonal principal de um a matriz x)
                - det(x) ( Retorna o determinante da matriz x)
                - zeros(m, n) ( Cria uma matriz considerada nula em uma soma com m linhas e n colunas, somente zeros)
                - ones(m, n) ( Cria uma matriz considerada nula em um produto com m linhas e m colunas → somente uns)
                - A(x, y) ( Elemento de uma matriz A situado na linha x e coluna y)
                - slicing A(x, 1:y) ( Pega uma fatia da matriz → linha x da coluna 1 até y → caso seja do inicio ao fim pode ser somente :)
                - cat(1, A, B) ( Concatena (junta em determinada orientação) as matrizes A e B desde que possuam a mesma dimenção no sentido da concatenação → vertical(1, A, B) n de colunas igual → horizontal(2, A, B) n de linha igual)
                - triu(A) ( Retorna a matriz triangular superior da matriz A )
                - tril(A) ( Retorna a matriz triangular inferior da matriz A)

    * Categorias de Linguagens de Dev. e Prog.

      • Linguagens Compiladas:
          São linguagens de Prog. / Dev. que compilam os arquivos das linhas de código num pacote que são usados em aplicações para hardwares específicios bem como o software por eles depurados, é mais rápida em resposta porém com menos portabilidade.


      • Linguagens Interpretadas:
          São Linguagens de Prog. / Dev. que compilam os arquivos de linha de código linha a linha seguindo o padrão de leitura de sintaxe de cada uma, porém são mais lentas na resposta e com uma portabilidade muito melhor.
